Product surface

The graph is the backend contract.

The UI is not a diagram pasted over a database. It is the control layer: every table, actor, policy, route, secret reference, deploy request, and audit event has a place in the graph.

Store

Model accounts, projects, graph versions, customer rows, backend instances, and field boundaries in the same place.

Hold

Keep secrets referenced, deploys reviewable, routes explicit, and generated artifacts tied back to the source graph.

Access

Human users, support, analysis agents, coding agents, and devices get scoped permissions instead of broad database access.
